# Cloud Secure Web Application (DevSecOps)

## Overview

This project is a cloud-based secure web application built on AWS using DevSecOps principles.

## Features

* User authentication
* Role-based access control
* CRUD operations
* Docker containerization
* CI/CD pipeline
* AWS deployment
* Monitoring and logging
* Security hardening

## Tech Stack

* Python Flask
* PostgreSQL
* Docker
* Nginx
* AWS
* GitHub Actions

## Project Structure

* docs/ → architecture diagrams and screenshots
* src/ → application source code
* infra/ → infrastructure files
* scripts/ → deployment and backup scripts
